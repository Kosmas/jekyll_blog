Problem
You want to install the GUI tools for MySQL in Mandriva 2010 but a quick search in the MySQL website doesn't list the Mandriva rpm files.

Solution
To install the tools in Mandriva 2010 run:
sudo urpmi mysql-query-browser mysql-administrator.
