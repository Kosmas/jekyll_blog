Problem
You want to use ActiveScaffold in your application, that you have developed using Rails 2.1, and you want to deploy it in Dreamhost, which at the moment has Rails 2.0.2.

Solution
Don't use the installation instructions in the main page from ActiveScaffold's website that uses the version in git:

git clone git://github.com/activescaffold/active_scaffold.git vendor/plugins/active_scaffold && rm -rf vendor/plugins/active_scaffold/.git

but use the older version that it's in svn and the installation instructions are:

./script/plugin install http://activescaffold.googlecode.com/svn/tags/active_scaffold
