Problem
Using the new will_paginate gem, you want to conditionally display the previous. next links, only if the number of records is bigger than the value set in the per_page configuration variable.

Solution
Instead of only using the:

<%= will_paginate @records %>

use the per_page to check before displaying the links, as in:

<% if @records.size > @records.per_page-1 %>
  <%= will_paginate @records %>
<% end %>
